Detailed Description

Defines class bs::shell_node_visitor_json

Defines class bs::shell_node_visitor_json to visit shell node structures. It is very useful to debug as it allows to understand the node structure better. This particular visitor converts the shell node structure into json, allowing visual understanding of the node structure.
