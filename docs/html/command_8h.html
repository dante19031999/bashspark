Detailed Description

Defines the bs::command interface and built-in shell commands.

This file provides the base class command for shell commands and several concrete commands such as echo, eval, getenv, getvar, setenv, setvar, and seq. Each command can be executed within a shell session and provides virtual error-handling methods to allow custom behavior, including internationalization.
